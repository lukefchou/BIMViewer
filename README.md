BIM浏览器，BIM引擎，WebBIM

1、支持格式

###1.1 常规3D格式

obj, 3ds, stl, ply, gltf, glb, off, 3dm, fbx, dae, wrl, 3mf, ifc, brep, step, iges, fcstd, bim, etc.

###1.2 BIM模型

(0) ifc：工业基础类，国际BIM标准

(1) rvt：Revit原生格式，支持最新版本Revit，带材质

(2) rvm：PDMS导出格式

(3) dgn：Bentley产品通用格式

2、技术方案
基于WebGL的三维可视化

3、BIM课件
包含BIM技术应用开发系列理论课件
